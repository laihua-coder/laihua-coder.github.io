Linux服务器安装nodejs
Linux
Linux
发布日期:   2019-05-28
文章字数:   441
阅读时长:   1 分

下载

下载源码，你需要在http://nodejs.org/下载最新的Nodejs版本，此处以node-v10.15.3-linux-x64.tar.xz版本为例。

解压文件

  1. 通过远程工具连接Linux服务并进入指定目录
cd /usr/local
  1. 将文件传送到该目录并解压文件
tar xvf node-v10.15.3-linux-x64.tar.xz
  1. 创建软链接，使node和npm命令全局有效。通过创建软链接的方法，使得在任意目录下都可以直接使用node和npm命令：
ln -s /root/node-v10.15.3-linux-x64/bin/node  /usr/local/bin/node
ln -s /root/node-v10.15.3-linux-x64/bin/npm  /usr/local/bin/npm
  1. 查看并复制路径
# cd usr/local/node-v10.15.3-linux-x64/bin/
# pwd

此时命令可以查看到路径如下:

/usr/local/node-v10.15.3-linux-x64/bin

node安装完毕，但是此时输入命令node -v还不能显示版本号，会提示node命令不存在，还需要配置文件。

全局环境变量安装

  1. 输入命令
vim /etc/profile
  1. 在vi 环境下 点击 i 进入插入状态，在export PATH的上一行添加如下内容 (环境变量中的内容 是以冒号分割的)
PATH=$PATH:/usr/local/node-v10.15.3-linux-x64/bin
  1. 编辑完成后按Esc键 然后输入 :wq 按回车保存退出。

  1. 退出vi ，执行source /etc/profile 可以是变量生效.

  2. 然后执行 echo $PATH ，看看输出内容是否包含自己添加的内容

验证安装

然后到任意目录下去执行一次执行node -v和npm -v

至此 nodejs在Linux服务器上安装完成
